## Writing Posts

Create a new markdown file in `src/content/blog` with frontmatter like:

```markdown
---
title: "Post Title"
date: 2024-12-23 23:47:00 +0000
categories: [Tech, Writing]
tags: [comms]
image: /images/posts/your-image.svg
description: "Short summary for previews."
---

Your content here.
```
